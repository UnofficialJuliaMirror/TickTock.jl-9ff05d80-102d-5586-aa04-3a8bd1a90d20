# TickTock

This module provides `tick()`, `tock()`, and `tok()` functions.

They're similar to the `tic()`, `toc()`, and `toq()` functions that you might find in MATLAB and
similar software.

## Example

```julia
julia-0.6> using TickTock

julia-0.6> tick()
0x0000146b7dcc5c23

julia-0.6> tock()
2 seconds
```
